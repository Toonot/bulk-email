# bulk-email
Program to send emails to recipients from a huge list in a performant way.

A simple client program to send bulk emails. Written in Go. It will send email message (from given HTML Template file) to a list of recipients (from given csv file) using multiple smtp server relays (from given serverconfiguration csv file).

The arguments are given through command line interface
